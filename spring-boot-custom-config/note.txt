Argument can be passed :
1. By setting config on application.properties or application.yml
2. By passwing parameter on run command : 
   $ ./mvnw spring-boot:run -Dserver.ip=192.168.12.1
  
3. When packaging into jar
   $ ./mvnw package -DskipTests=true
   $ java -jar target/spring-boot-config-0.0.1-SNAPSHOT.jar --server.ip=192.168.12.1
